# mrm-relay-server
A Medical Records Management relay server

### Running the project

#### Step 1: Install dependencies
     run npm i
 
#### Step 2: Add environment variables
 
    BASE_URL=
#### Step 3: Run the server
Development mode:

    npm run start:dev

Production mode:

    npm run start
    
#### Running in Docker (still WIP)

    docker-compose up -d
